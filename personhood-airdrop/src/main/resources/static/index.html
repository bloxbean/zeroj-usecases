🪂 Sybil-Resistant Airdrop

Each personhood credential can claim ADA once per epoch. The ZK proof commits to a nullifier = Poseidon(personhoodId, epoch). Cardano enforces nullifier uniqueness on-chain via the minted claim NFT (one NFT per nullifier per policy ID). Same person trying again in the same epoch produces the same nullifier → tx fails to build.
